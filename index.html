Probando framework de BackboneJS+Twitter Bootstrap sobre Heroku

Simpleza y elegancia. Facilitando el desarrollo y permitiendo empujar los limites creativos cada vez mas. Una vez que las piezas iniciales estan en su lugar, la aplicacion empieza a tomar mas forma de forma muy rapida y constante.

El siguiente parrafo llega a usted gracias a Backbone:

Interfaz al usuario:

Bootstrap para una presentacion agradable y de facil mantenimiento y extensibilidad

Manejo de datos:

Backbone siguiendo el modelo MVC que permite intercambiar cada pieza separadamente y mantener la aplicacion funcional

Publicacion:

Heroku, servidores elasticos en nube de facil acceso y alta disponibilidad

Repositorio:

Git por seguridad del avance del proyecto y permitir colaboracion distribuida desde el dia 1
